Guide de déploiement PWA · v3.2
📱 Scanner pour tester sur téléphone
Scanner avec l'appareil photo du téléphone
🚀 Mettre en ligne en 10 minutes (GitHub Pages)
1
Créer un compte gratuit sur github.com
2
Créer un nouveau dépôt (repository) nommé te-hoe-temata
3
Uploader ces 4 fichiers :
te_hoe_app_v3_2.html
manifest.json
sw.js
icon-192.png + icon-512.png
4
Dans Settings → Pages → Branch: main → Save
5
Ton URL sera : ton-pseudo.github.io/te-hoe-temata/te_hoe_app_v3_2.html
Colle-la dans le champ QR ci-dessus → Scanner → Installer l'app
6
Android : Chrome → bannière "Ajouter à l'écran d'accueil" apparaît automatiquement
iPhone : Safari → bouton partage → "Sur l'écran d'accueil"
💳 Activer les Tips Stripe
1. Créer un compte sur stripe.com (gratuit, 5 min)

2. Créer un Payment Link
Dashboard → Payment Links → Nouveau → Produit "Tip Te Hoe" → Prix variable

3. Copier l'URL du type https://buy.stripe.com/xxxx

4. Dans te_hoe_app_v3_2.html, remplacer ligne :
const STRIPE_PAYMENT_LINK = null;
par :
const STRIPE_PAYMENT_LINK = 'https://buy.stripe.com/xxxx';

✅ Apple Pay et Google Pay s'activent automatiquement via Stripe
💶 Frais Stripe : ~1.5% + 0.25€ par transaction
